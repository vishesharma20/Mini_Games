# Tic Tac Toe

A tic tac toe game, but you can swap X and O for literally anything 
including, apparently, Shawn and Justin. Why not, right? 😄

##Live Demo
https://vishesharma20.github.io/Mini_Games/tic%20tac%20toe/

## What's in it

- **vs Computer** with adjustable difficulty (easy / medium / hard — hard is
  unbeatable, don't say I didn't warn you)
- **vs Friend** pass-and-play mode
- A few built-in icon sets (spooky, cute, space, nature) or just upload
  your own images for each player — used it to pit Shawn against Justin in
  pass & play mode for fun
